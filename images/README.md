# App Screenshots

Place your app screenshots in this folder with the following names:

## Required Images:
- `app1-screenshot.png` - HealthTracker Pro screenshot
- `app2-screenshot.png` - PayEase Wallet screenshot  
- `app3-screenshot.png` - TaskFlow Manager screenshot

## Image Guidelines:
- **Format:** PNG or JPG
- **Recommended size:** 300x600px (mobile app aspect ratio)
- **Max file size:** 500KB for optimal loading
- **Quality:** High resolution but optimized for web

## Tips for Best Results:
1. Use actual app screenshots from your device
2. Consider using device frames (iPhone/Android mockups)
3. Ensure screenshots show key features of your apps
4. Compress images using tools like TinyPNG for faster loading
5. Use consistent styling across all screenshots

## Adding More Apps:
To add more app screenshots:
1. Add new image file (e.g., `app4-screenshot.png`)
2. Copy an app card in `index.html` 
3. Update the `src` attribute to point to your new image
4. Update app title, description, and tech tags

## Alternative Formats Supported:
- PNG (recommended for UI screenshots)
- JPG (good for photos/illustrations)
- WebP (modern format, smaller file sizes)
